
# 🧠 Cognivex+

Cognivex+ is a **friendly AI tutor system** designed for students to help them learn, solve doubts, and improve productivity using a chat-based interface.

---

## 🚀 Features

- 💬 AI Chat Assistant
- 📚 Student-friendly explanations
- 💻 Coding help & debugging support
- ⚡ Fast responses using Groq LLM
- 🌐 Web-based chat interface
- 🧠 Basic conversation memory (in progress)

---

## 🏗️ Tech Stack

### Backend
- Python
- FastAPI
- Groq API (LLM)

### Frontend
- HTML
- CSS
- JavaScript
